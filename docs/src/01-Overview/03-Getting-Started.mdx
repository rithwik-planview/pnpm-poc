# Getting Started

Follow these steps to set up your development environment:

## Initial Setup

1. Install dependencies:

```bash
pnpm install
```

2. Compile localization files:

```bash
pnpm -r i18n
```

3. Build all packages:

```bash
pnpm -r build
```

## Development

### Running Tests

```bash
pnpm -r test
```

### Starting Storybook

```bash
pnpm -F @unity/storybook dev
```

## Feature Development

1. Start development server for a feature (example: `workplan-import`):

```bash
pnpm -F "@unity/*.workplan-import" mf-dev
```

2. Start production preview server:

```bash
# Build and deploy the feature
pnpm -F @unity/features.workplan-import deploy-package

# Start local server
pnpm serve:local-build-drop

# Build and preview the app
pnpm -F @unity/apps.workplan-import build
pnpm -F @unity/apps.workplan-import preview
```
