# Getting Started

1. Install dependencies:

```bash
pnpm install
```

2. Compile localization files:

```bash
pnpm -r i18n
```

3. Build all packages:

```bash
pnpm -r build
```

4. Run tests:

```bash
pnpm -r test
```

5. Start Storybook:

```bash
pnpm -F @unity/storybook dev
```

6. Start development server for a feature (example: `workplan-import`):

```bash
pnpm -F "@unity/*.workplan-import" mf-dev
```

7. Start production preview server for a feature (example: `workplan-import`):

```bash
pnpm -F @unity/features.workplan-import deploy-package
pnpm serve:local-bucket
pnpm -F @unity/apps.workplan-import build
pnpm -F @unity/apps.workplan-import preview
```
